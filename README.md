# 🇪🇺 EU with Mojtaba — PWA

Application de révision gamifiée pour le concours SAEO 2026 (Questions européennes).

## 📦 Fichiers inclus

```
eu-mojtaba-pwa/
├── index.html      ← L'application complète
├── manifest.json   ← Config PWA (nom, icônes, couleurs)
├── sw.js           ← Service Worker (cache offline)
└── README.md       ← Ce fichier
```

## 🚀 Installation sur mobile

### Option 1 — Hébergement gratuit (recommandé)

1. Va sur **https://netlify.com** (gratuit)
2. Crée un compte
3. Glisse-dépose le dossier `eu-mojtaba-pwa` sur Netlify
4. Tu obtiens une URL type `https://eu-mojtaba.netlify.app`

### Option 2 — GitHub Pages (gratuit)

1. Crée un repo GitHub
2. Upload les fichiers
3. Active GitHub Pages dans Settings
4. URL : `https://tonnom.github.io/eu-mojtaba`

---

## 📱 Installer sur iPhone/iPad

1. Ouvre l'URL dans **Safari** (obligatoire, pas Chrome)
2. Appuie sur le bouton **Partager** ⬆️
3. Sélectionne **"Sur l'écran d'accueil"**
4. Appuie sur **Ajouter**
5. L'app apparaît sur ton écran d'accueil comme une vraie app !

## 📱 Installer sur Android

1. Ouvre l'URL dans **Chrome**
2. Une bannière apparaît en bas : **"Installer EU with Mojtaba"**
3. Appuie sur **Installer**
4. L'app s'installe comme une vraie application !
